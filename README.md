Sistema de Gestão Laboratorial (LACIF)

Sobre o Projeto
Este projeto tem como objetivo desenvolver um sistema para gerenciamento de informações laboratoriais, auxiliando no controle, organização e consulta de dados. A aplicação foi pensada para simular um ambiente de um laboratório do Instituto de Zootecnia da UFRRJ.

Tecnologias Utilizadas
- Java
- JavaFX
- MySQL

Funcionalidades
- Cadastro de dados laboratoriais
- Consulta e listagem de informações
- Atualização de registros
- Exclusão de dados
- Organização e estruturação das informações

Como Executar
Clone o repositório:
- git clone https://github.com/lucasBnunes4/Projeto-LACIF.git
- Abra o projeto em uma IDE
- Execute conforme a configuração do projeto
- Acesse a aplicação no navegador
